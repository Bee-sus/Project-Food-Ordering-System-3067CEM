Description: 
This project is about a food ordering system that allows users to browse through various food options, manage food orders, and view order details with its cost. 

Instructions: 
1. Make sure the database file is in the same folder as the source code.
2. Change the picture file path of the source code according to where the image is saved on the tester's device.
3. Run the code in PyCharm.

Libraries to be installed:
1. pillow 
2. fpdf
